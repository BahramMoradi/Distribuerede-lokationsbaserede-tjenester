Main scenario
  1. User start the application and navigate to record route.
  2. following items are presented in the Record route.
    1. Record button.
    2. Google map.
  3. User click on Record button.
  4. Application records user location in the defined interval in Settings activity .
  5. Application stores the user location in the local database.
  6. Application sends user location to remote service as bulk data with fixed interval defined by user in the settings Activity.
  7. Remote service sends acknowledgement to the application.
  8. Application remove the send data from local database

Related information
