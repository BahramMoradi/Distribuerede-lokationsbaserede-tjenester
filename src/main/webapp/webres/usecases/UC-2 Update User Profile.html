Scope & Level
 User Profile
Goal in context
 Update user profile
Preconditions
 User has previously created a profile , there is network connection, web service is running.
Successful outcome
 User's profile updated successfully
Failure Outcomes
 Failure
 Outcome
 Condition leading to outcome
Network connection failure
 No network connection is available 
User has not enabled mobile network
No route to host
No route to the host(url)

Primary actor
 User
Secondary actors
 System
Main scenario
  1. User start the application
  2. User navigate to the User Profile page from the menu
  3. Following items are presented to the user.
    • Labels and their corresponding  text fileds:
      • Name
      • Phone
      • E-Mail
      • Description
    • Buttons:
      • Save
      • Update
      • delete
  4. User change the information in the text fields.
  5. User click update.
  6. System sends user information to web service.
  7. System shows "Profile updated successfully" to the user.
