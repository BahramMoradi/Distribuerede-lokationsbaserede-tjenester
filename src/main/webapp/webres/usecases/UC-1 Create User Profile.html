Main scenario
 1. User start the application by clicking on the app.
2. Application start create profile activity.
3. User enters following information
  • Username
  • E-mail:
  • Phone number
  • Description
4. User click create profile.
5. The system send information to the user resource web service
6. Web service save user information.
7. Web service return the result to the user
